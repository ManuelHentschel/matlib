Class Data Set

Description

A small artificial data set used to illustrate statistical concepts.

Usage

data("class")

Format

A data frame with 15 observations on the following 4 variables.

sex

a factor with levels F M

age

a numeric vector

height

a numeric vector

weight

a numeric vector

Examples

data(class)
plot(class)
