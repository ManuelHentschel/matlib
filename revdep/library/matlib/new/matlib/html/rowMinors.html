rowMinors {matlib}R Documentation

Row Minors of A[i,]

Description

Returns the vector of minors of row i of the square matrix A

Usage

rowMinors(A, i)

Arguments

A

a square matrix

i

row index

Value

a vector of the minors of A[i,]

Author(s)

Michael Friendly

See Also

Other determinants: Det(), adjoint(), cofactor(), minor(), rowCofactors()

Examples

M <- matrix(c(4, -12, -4,
              2,   1,  3,
             -1,  -3,  2), 3, 3, byrow=TRUE)
minor(M, 1, 1)
minor(M, 1, 2)
minor(M, 1, 3)
rowMinors(M, 1)
